# 第二课：打开 Pi 聊天历史

> 目标：学会员工在 Pi 输入框为空时，连续按两次 `ESC`，打开聊天历史并恢复之前的会话。

---

## 本课目标

完成后，你应该能做到：

1. 知道什么时候可以打开聊天历史；
2. 知道必须先让输入行保持空白；
3. 会连续按两次 `ESC` 打开历史；
4. 会从历史里选择并恢复之前的会话。

---

## 第 1 步：进入 Pi

在终端输入：

```bash
pi
```

进入 Pi 对话界面。

---

## 第 2 步：确保输入行是空的

先看最下面的输入行。

如果里面有任何文字，先删掉，确保输入行是空白状态。

> 只有在输入行为空的时候，这个快捷操作才适用。

---

## 第 3 步：连续按两次 `ESC`

当输入行为空时：

1. 按一次 `ESC`；
2. 紧接着再按一次 `ESC`。

也就是：

```text
ESC ESC
```

这样会打开 Pi 的聊天历史。

---

## 第 4 步：选择历史会话

打开聊天历史后，你可以：

- 查看之前的会话；
- 选择一条历史会话；
- 继续之前没做完的任务。

如果你想接着昨天的 PRD 继续写，就从这里恢复，不需要重新开一个新对话再重复解释背景。

---

## 什么时候最有用

这个功能特别适合下面几种情况：

- 昨天写了一半，今天继续；
- 想找回之前生成过的 `prd.md` 讨论过程；
- 想继续上一次的需求澄清；
- 想查看自己之前让 Pi 做过什么。
